AI Chatbot Database Integration

A smart and interactive chatbot that simplifies database queries by leveraging Qwen-2.5-32B, a large language model from Alibaba Cloud. Built with Flask, SQLite, and Tailwind CSS, this application allows users to retrieve expense data using natural language instead of writing SQL queries manually. The chatbot intelligently generates SQL commands, fetches relevant data, and presents the results in a structured table format. Designed for efficiency and ease of use, this project showcases the power of AI in streamlining database interactions.
